Quaff The Shelf

Quaff The Shelf is the final project undertaken with colleagues at Northcoders. It is a mobile and web application which pairs book with drinks suggestions.
The app allows the user to create an account, scan in a book with barcode (or by adding with ISBN) and add it to their collection. The user can then upvote or downvote drinks pairing suggestions or suggest a pairing of their own.

Quaff The Shelf was built with React Native on the front end and a Firebase non-relational database at the backend. Books are accessed with calls to the Open Library API. When finished, the team presented the app at the Northcoders graduation streamed on Youtube:

Games Emporium

Games Emporium is a web application which combines the front end and back end projects undertaken at Northcoders. The application is a database of board games reviews.
The user can search for reviews by category and sort the results. They can also vote on reviews, and add and delete their own comments.
The front end was built with React and back end with a javascript API and PSQL database.

Losing and Finding a Home

Losing and Finding a Home is an academic website built for the purpose of public engagement with academic research findings. The site is built in Wordpress using PHP.
